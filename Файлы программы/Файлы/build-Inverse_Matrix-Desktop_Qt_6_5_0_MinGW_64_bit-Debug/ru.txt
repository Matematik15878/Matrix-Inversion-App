Нахождение обратной матрицы
Выберите метод:
Нажмите, чтобы произвольно заполнить
Выберите границы:
Шагов при решении сделано:
Нажмите, чтобы сохранить матрицу
Поехали!
Нажмите, чтобы загрузить матрицу
Выберите размер:
Исходная матрица A
Обратная матрица B
Помни!
1) A * B = E.
2) A * B = B * A. 
3) Элементы главной диагонали могут = 0.
4) Обращаются только квадратные матрицы.
5) Матрица необратима, если определитель = 0.
Ошибка! Матрицу невозможно обратить: либо она необратима, либо определитель не найден!
